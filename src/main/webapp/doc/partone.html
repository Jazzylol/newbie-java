多态

  1. 多态是什么意思？

  2. 多态在程序中的体现？父类的引用或者接口的引用 实际指向的子类的对象. Father f = new Son1(). Father f = new Son2(), Animal a = new Dog(). Animal b = new Cat()

  3. 多态情况下，向下转型指的是 将父类强行转成子类，例如 Animal a = new Dog(); Dog dog = (Dog) a;

    因为已经事先知道a指向的 dog才可以向下转型，如果不知道需要使用关键字 instanceof 判断， if（ a instanceof Dog) Dog d = (Dog) a. 如果a不是dog类型则会报错

  4. 向下转型的好处是可以使用子类的特有方法，一般在多态的情况下 Father f = new Son(),这种 使用f调用方法的时候只能使用 Father类中定义的方法，在子类Son中定义的特有方法是无法调用到的，向下转型过以后才可以调用Son的特有方法。

  5. 在多态的情况下有一种特殊情况，如果Animal a = new Dog(). 如果父类和子类存在相同的成员变量的时候，JAVA在编译的时候都是看 等号左面即父类中该变量是否有语法错误，如果没有编译通过，当运行时（程序中使用该变量）那么依旧是检查等号左面的即父类中该变量使用是否正确。但是对于子父类中包含相同的方法（复写）的时候，java在编译的时候检查的是等号左面即父类中该方法是否语法正确，但是在运行的时候（程序调用该方法）是调用的等号右面即子类中的方法。如果不是子父类中相同的变量或者方法，统一编译运行全部是调用父类相关的变量和方法

  6. 内部类是什么？如何定义？内部类就是 类中类，在类中继续定义类。内部类的好处是什么？（不给外部其他类访问的权限） 内部类使用外部类的变量，方法的时候如何使用？外部类使用内部类的变量、方法的时候如何使用
